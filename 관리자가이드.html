강의 관리부터 결제 처리까지 모든 기능을 쉽게 사용할 수 있습니다

  • 시작하기
  • 대시보드
  • 회원 관리
  • 강의 관리
  • 결제 관리
  • 환불 관리
  • 공지/문의
  • 자주 묻는 질문

시작하기

관리자 페이지 접속 방법과 주요 메뉴를 안내합니다

관리자 페이지 접속하기

1 웹사이트 접속

웹 브라우저(크롬, 사파리, 엣지 등)를 열고 웹사이트 주소를 입력합니다.

2 로그인

우측 상단의 로그인 버튼을 클릭하여 관리자 계정으로 로그인합니다.

  • 이메일 주소와 비밀번호를 입력합니다
  • 또는 Google, Kakao, Naver 계정으로 로그인할 수 있습니다

3 관리자 페이지 진입

로그인 후, 웹사이트 주소 뒤에 /admin을 붙여 접속합니다.

예시: yourdomain.com/admin

관리자 권한 확인

관리자 페이지가 보이지 않는다면 아직 관리자 권한이 부여되지 않은 것입니다. 개발팀에 문의하여 관리자 권한을 부여받으세요.

관리자 메뉴 구성

대시보드

전체 현황을 한눈에 확인하는 메인 화면입니다. 매출, 회원 수, 수강 통계를 볼 수 있습니다.

회원 관리

가입한 회원들을 확인하고, 검색하고, 관리할 수 있습니다.

강의 관리

강의를 만들고, 수정하고, 영상을 업로드하는 기능입니다.

결제 관리

결제 내역을 확인하고, 무통장입금을 승인/거절합니다.

환불 관리

회원들의 환불 요청을 처리합니다.

공지사항

회원들에게 알릴 공지사항을 작성하고 관리합니다.

1:1 문의

회원들의 문의에 답변합니다.

통계

매출, 회원, 강의 통계를 상세하게 확인합니다.

대시보드 사용하기

플랫폼의 전체 현황을 한눈에 파악할 수 있는 메인 화면입니다

주요 통계 확인

대시보드 상단에 4개의 주요 지표가 표시됩니다:

  • 전체 회원: 플랫폼에 가입한 총 회원 수와 최근 신규 가입자 수
  • 전체 강의: 현재 운영 중인 강의 개수
  • 전체 수강 신청: 총 수강 신청 건수와 최근 수강 신청 수
  • 전체 수익: 총 매출액과 최근 매출

기간 선택하기

화면 상단의 드롭다운 메뉴에서 통계 기간을 선택할 수 있습니다:

  • 최근 7일
  • 최근 30일 (기본값)
  • 최근 90일
  • 최근 1년

결제 현황

완료된 결제 건수와 무통장입금 대기 중인 건수를 실시간으로 확인할 수 있습니다.

무통장입금 알림

무통장입금 대기 건이 있으면 숫자가 표시됩니다. "결제 관리" 메뉴로 이동하여 입금 확인 후 승인해 주세요.

그래프로 보는 추이

  • 회원 가입 추이: 최근 7일간 일별로 몇 명이 가입했는지 막대 그래프로 표시
  • 일별 수익 추이: 최근 7일간 하루에 얼마씩 매출이 발생했는지 막대 그래프로 표시

인기 강의 Top 5

매출이 높은 순서대로 상위 5개 강의가 표시됩니다. 각 강의의 수강생 수, 구매 건수, 총 수익을 확인할 수 있습니다.

최근 활동

가장 최근에 발생한 수강 신청과 결제 내역을 확인할 수 있습니다. 회원 이름을 클릭하면 해당 회원의 상세 정보를 볼 수 있습니다.

회원 관리하기

가입한 모든 회원을 확인하고 관리할 수 있습니다

회원 목록 보기

회원 목록에서 다음 정보를 확인할 수 있습니다:

  • 회원 이름
  • 이메일 주소
  • 역할 (학생 또는 관리자)
  • 가입일
  • 수강 중인 강의 수

회원 검색하기

이름 또는 이메일로 검색

  1. 화면 상단의 검색창을 찾습니다
  2. 찾고자 하는 회원의 이름 또는 이메일을 입력합니다
  3. 자동으로 검색 결과가 표시됩니다

회원 상세 정보 확인하기

회원 정보 자세히 보기

  1. 회원 목록에서 확인하고 싶은 회원의 이름을 클릭합니다
  2. 상세 페이지에서 다음 정보를 확인할 수 있습니다:
    • 기본 정보: 이름, 이메일, 전화번호, 가입일
    • 수강 정보: 수강 중인 강의 목록과 진도율
    • 결제 내역: 구매한 강의와 결제 금액
    • 등록 기기: 현재 사용 중인 기기 목록

회원을 관리자로 지정하기

다른 사람에게 관리자 권한 부여

  1. 회원 목록에서 관리자로 지정할 회원을 찾습니다
  2. 회원 이름을 클릭하여 상세 페이지로 이동합니다
  3. 역할 변경 버튼을 클릭합니다
  4. "관리자(ADMIN)"를 선택합니다
  5. "확인" 버튼을 클릭합니다
주의하세요!

관리자 권한을 부여하면 해당 회원은 모든 관리 기능을 사용할 수 있게 됩니다. 신뢰할 수 있는 사람에게만 권한을 부여하세요.

회원 기기 관리하기

각 회원은 최대 2개의 기기(PC, 스마트폰, 태블릿 등)에서만 강의를 시청할 수 있습니다.

회원의 기기 삭제하기

  1. 회원 상세 페이지에서 "등록 기기" 섹션을 찾습니다
  2. 삭제할 기기의 삭제 버튼을 클릭합니다
  3. 확인 메시지에서 "확인"을 클릭합니다

기기를 삭제하면 해당 기기에서는 더 이상 영상을 볼 수 없고, 회원이 새로운 기기를 등록할 수 있게 됩니다.

기기 제한 안내

회원이 "최대 기기 수를 초과했다"는 메시지를 받으면, 관리자 페이지에서 사용하지 않는 기기를 삭제해 주시면 됩니다.

강의 관리하기

강의를 만들고 관리할 수 있습니다

강의 목록 확인하기

현재 등록된 모든 강의를 확인할 수 있습니다. 각 강의의 제목, 가격, 강사명, 공개 여부, 영상 개수, 수강생 수가 표시됩니다.

새 강의 만들기

1 강의 생성 시작

강의 목록 화면에서 새 강의 만들기 버튼을 클릭합니다.

2 기본 정보 입력

다음 정보를 입력합니다:

  • 강의 제목: 학생들에게 보여질 강의명
  • 강의 설명: 강의 소개 내용
  • 가격: 강의 판매 금액 (원 단위로 입력)
  • 강사명: 강의를 진행하는 강사 이름
  • 강사 소개: (선택) 강사에 대한 간단한 소개

3 썸네일 이미지 업로드

강의를 대표하는 이미지를 업로드합니다 (선택사항이지만 권장):

  1. "파일 선택" 버튼을 클릭합니다
  2. 컴퓨터에서 이미지 파일을 선택합니다
  3. 이미지가 미리보기로 표시됩니다

4 공개 설정

  • 비공개: 학생들에게 보이지 않습니다 (강의 준비 중일 때 선택)
  • 공개: 학생들이 강의 목록에서 보고 구매할 수 있습니다

처음에는 비공개로 만들고, 영상을 모두 업로드한 후 공개로 변경하는 것을 추천합니다.

5 저장하기

저장 버튼을 클릭하여 강의를 생성합니다.

강의에 영상 추가하기

강의를 만든 후에는 영상을 추가해야 합니다.

Vimeo 영상 필요

이 플랫폼은 Vimeo에 업로드된 영상을 사용합니다. 먼저 Vimeo에 영상을 업로드한 후, 영상 주소(URL)를 복사해 오세요.

Vimeo 주소 예시: https://vimeo.com/123456789

1 강의 상세 페이지로 이동

  1. 강의 목록에서 영상을 추가할 강의의 제목을 클릭합니다
  2. 강의 상세 페이지가 열립니다

2 영상 추가하기

  1. 영상 추가 버튼을 클릭합니다
  2. 다음 정보를 입력합니다:
    • 영상 제목: 강의 차시명 (예: "1강. 파이썬 소개")
    • 영상 설명: (선택) 해당 영상의 간단한 설명
    • Vimeo URL: Vimeo 영상 주소를 붙여넣기
    • 순서: 영상의 순서 번호 (1, 2, 3...)
  3. 추가 버튼을 클릭합니다

3 미리보기 영상 설정 (선택)

미리보기로 설정한 영상은 강의를 구매하지 않은 사람도 무료로 볼 수 있습니다.

  1. 영상을 추가할 때 미리보기 체크박스를 선택합니다
  2. 또는 이미 추가된 영상을 수정하여 미리보기로 설정할 수 있습니다
미리보기 활용 팁

첫 번째 강의나 강의 소개 영상을 미리보기로 설정하면 학생들이 강의 스타일을 미리 확인하고 구매할 수 있어 구매율이 높아집니다.

영상 순서 변경하기

드래그로 순서 바꾸기

  1. 강의 상세 페이지의 영상 목록에서
  2. 영상을 마우스로 클릭한 채로 드래그하여 원하는 위치로 이동합니다
  3. 순서가 자동으로 저장됩니다

영상 수정 및 삭제

영상 수정하기

  1. 영상 목록에서 수정할 영상을 찾습니다
  2. 수정 버튼을 클릭합니다
  3. 제목, 설명, URL 등을 변경합니다
  4. "저장" 버튼을 클릭합니다

영상 삭제하기

  1. 영상 목록에서 삭제할 영상을 찾습니다
  2. 삭제 버튼을 클릭합니다
  3. 확인 메시지에서 "확인"을 클릭합니다
영상 삭제 주의!

영상을 삭제하면 학생들의 시청 기록과 진도율도 모두 삭제됩니다. 신중하게 결정하세요.

강의 자료 업로드하기

PDF, 문서, 이미지 등의 보충 자료를 강의에 첨부할 수 있습니다.

자료 추가하기

  1. 강의 상세 페이지에서 "강의 자료" 섹션을 찾습니다
  2. 파일 선택 버튼을 클릭합니다
  3. 컴퓨터에서 업로드할 파일을 선택합니다
  4. 업로드 버튼을 클릭합니다
  5. 업로드가 완료되면 자료 목록에 표시됩니다

학생들은 강의 페이지에서 자료를 다운로드할 수 있습니다.

강의 수정하기

강의 정보 변경하기

  1. 강의 목록에서 수정할 강의를 클릭합니다
  2. 강의 상세 페이지에서 수정 버튼을 클릭합니다
  3. 제목, 설명, 가격, 공개 여부 등을 변경합니다
  4. "저장" 버튼을 클릭합니다

강의 삭제하기

강의 삭제 전 반드시 확인!

강의를 삭제하면 다음 내용이 모두 삭제됩니다:

  • 모든 영상
  • 모든 자료
  • 학생들의 수강 기록
  • 결제 내역

권장: 이미 학생들이 구매한 강의는 삭제하지 말고 "비공개"로 전환하세요.

결제 관리하기

모든 결제 내역을 확인하고 무통장입금을 처리할 수 있습니다

결제 내역 확인하기

결제 목록에서 다음 정보를 확인할 수 있습니다:

  • 주문 번호
  • 구매자 이름과 이메일
  • 강의 제목
  • 결제 금액
  • 결제 수단 (카드 결제 또는 무통장입금)
  • 결제 상태 (완료, 대기, 취소)
  • 결제일시

무통장입금 승인하기

학생이 무통장입금으로 결제하면 관리자가 입금을 확인하고 승인해야 수강이 시작됩니다.

1 무통장입금 목록 확인

  1. 좌측 메뉴에서 "무통장입금 관리" 또는 "결제 관리 > 무통장입금"을 클릭합니다
  2. 입금 대기 중인 목록이 표시됩니다

2 입금 확인

다음 정보를 확인합니다:

  • 입금자명: 학생이 입력한 입금자 이름
  • 결제 금액: 입금해야 할 금액
  • 입금 예정일: 학생이 선택한 입금 날짜

실제로 은행 계좌에 입금이 되었는지 확인합니다.

3 입금 승인

  1. 입금이 확인되면 승인 버튼을 클릭합니다
  2. 확인 메시지에서 "확인"을 클릭합니다
  3. 승인이 완료되면:
    • 학생이 즉시 강의를 수강할 수 있게 됩니다
    • 영수증이 자동으로 발행됩니다
    • 결제 상태가 "완료"로 변경됩니다

무통장입금 거절하기

입금이 확인되지 않거나 문제가 있는 경우 거절할 수 있습니다.

입금 거절 절차

  1. 무통장입금 목록에서 거절할 건을 찾습니다
  2. 거절 버튼을 클릭합니다
  3. 거절 사유를 입력합니다 (예: "입금 확인 불가", "금액 불일치" 등)
  4. "확인" 버튼을 클릭합니다
  5. 학생에게 거절 사유가 알림으로 전달됩니다

영수증 확인하기

결제가 완료되면 영수증이 자동으로 발행됩니다. 학생들은 마이페이지에서 영수증을 확인하고 출력할 수 있습니다.

세금계산서 관리하기

사업자인 학생이 세금계산서를 요청하면 관리자가 처리합니다.

세금계산서 발행 요청 확인

  1. 좌측 메뉴에서 "세금계산서 관리"를 클릭합니다
  2. 발행 요청 목록에서 다음 정보를 확인합니다:
    • 사업자등록번호
    • 상호명
    • 대표자명
    • 사업장 주소
    • 이메일 (세금계산서 발송용)
    • 연락처
  3. 세금계산서 발행 시스템(별도)을 통해 세금계산서를 발행합니다
  4. 발행이 완료되면 발행 완료 버튼을 클릭합니다
세금계산서 발행 안내

이 시스템은 세금계산서 발행 요청 정보를 수집하는 기능을 제공합니다. 실제 세금계산서 발행은 별도의 전자세금계산서 시스템(국세청 홈택스, 전자세금계산서 발행 업체 등)을 이용하셔야 합니다.

환불 관리하기

학생들의 환불 요청을 확인하고 처리할 수 있습니다

환불 요청 확인하기

환불 요청 목록에서 다음 정보를 확인할 수 있습니다:

  • 요청자 이름
  • 강의명
  • 환불 사유
  • 환불 금액
  • 환불 계좌 정보 (무통장입금으로 결제한 경우)
  • 요청일
  • 처리 상태

환불 승인하기

1 환불 사유 검토

환불 요청 내역을 클릭하여 상세 정보를 확인하고, 환불 사유가 정당한지 검토합니다.

2 환불 승인

  1. 승인 버튼을 클릭합니다
  2. 확인 메시지에서 "확인"을 클릭합니다
  3. 환불 계좌 정보를 확인합니다

3 실제 환불 처리

무통장입금으로 결제한 경우:

  • 화면에 표시된 계좌번호로 환불 금액을 송금합니다

카드로 결제한 경우:

  • 토스페이먼츠 관리자 페이지에 로그인합니다
  • 해당 결제 건을 찾아 환불 처리를 진행합니다

4 환불 완료 처리

  1. 실제 환불이 완료되면 환불 목록으로 돌아옵니다
  2. 해당 건의 완료 처리 버튼을 클릭합니다
  3. 학생의 수강 권한이 자동으로 취소됩니다

환불 거절하기

환불 정책에 따라 환불이 불가능한 경우 거절할 수 있습니다.

환불 거절 절차

  1. 환불 요청 목록에서 거절할 건을 선택합니다
  2. 거절 버튼을 클릭합니다
  3. 거절 사유를 상세히 입력합니다 (예: "강의 진도율 50% 이상으로 환불 불가")
  4. "확인" 버튼을 클릭합니다
  5. 학생에게 거절 사유가 알림으로 전달됩니다
환불 정책 예시
  • 강의 시청 시작 전: 100% 환불
  • 강의 진도율 10% 미만: 80% 환불
  • 강의 진도율 10% 이상: 환불 불가
  • 결제일로부터 7일 이내: 전액 환불 (단, 시청 시작 전)

환불 정책은 서비스 특성에 맞게 조정하여 운영하세요.

공지사항 및 문의 관리하기

공지사항을 작성하고 학생들의 문의에 답변할 수 있습니다

공지사항 작성하기

학생들에게 중요한 소식을 알릴 때 공지사항을 사용합니다.

1 공지사항 메뉴 이동

좌측 메뉴에서 공지사항을 클릭합니다.

2 새 공지 작성

  1. 새 공지 작성 버튼을 클릭합니다
  2. 제목을 입력합니다
  3. 본문을 작성합니다 (글꼴, 크기, 색상 변경 가능, 이미지 삽입 가능)
  4. (선택) 첨부파일이 있으면 "파일 선택" 버튼으로 업로드합니다

3 상단 고정 설정 (선택)

중요한 공지는 상단 고정 체크박스를 선택하면 항상 목록 맨 위에 표시됩니다.

4 게시하기

게시 버튼을 클릭하면 즉시 학생들에게 공지가 표시됩니다.

공지사항 수정 및 삭제

공지 수정하기

  1. 공지사항 목록에서 수정할 공지를 클릭합니다
  2. 수정 버튼을 클릭합니다
  3. 내용을 변경하고 "저장" 버튼을 클릭합니다

공지 삭제하기

  1. 공지사항 상세 페이지에서 삭제 버튼을 클릭합니다
  2. 확인 메시지에서 "확인"을 클릭합니다

1:1 문의 답변하기

학생들이 질문을 남기면 관리자가 답변을 작성할 수 있습니다.

1 문의 목록 확인

  1. 좌측 메뉴에서 1:1 문의를 클릭합니다
  2. 답변 대기 중인 문의가 표시됩니다

2 문의 내용 확인

  1. 답변할 문의의 제목을 클릭합니다
  2. 학생이 작성한 문의 내용을 확인합니다

3 답변 작성

  1. 하단의 "답변 작성" 입력창에 답변을 작성합니다
  2. 필요시 글꼴, 이미지 등을 활용하여 작성합니다
  3. 답변 등록 버튼을 클릭합니다

4 답변 완료

답변이 등록되면:

  • 문의 상태가 "답변 완료"로 변경됩니다
  • 학생이 답변을 확인할 수 있습니다
빠른 답변 팁

자주 묻는 질문은 미리 답변 내용을 문서로 만들어 두고, 복사해서 붙여넣으면 신속하게 답변할 수 있습니다.

문의 필터링

상단의 필터 메뉴를 사용하여 문의를 분류할 수 있습니다:

  • 전체: 모든 문의
  • 답변 대기: 아직 답변하지 않은 문의만
  • 답변 완료: 답변이 완료된 문의만

통계 확인하기

플랫폼의 성과를 상세하게 분석할 수 있습니다

매출 통계

  • 일별, 주별, 월별 매출 추이 그래프
  • 결제 수단별 매출 비율 (카드 vs 무통장입금)
  • 강의별 매출 순위
  • 평균 구매 금액

회원 통계

  • 신규 가입자 수 추이
  • 전체 회원 수 및 증가율
  • 가입 경로별 분포 (이메일, Google, Kakao, Naver)
  • 활성 사용자 수

강의 통계

  • 강의별 수강생 수
  • 강의별 평균 진도율
  • 완강률 (끝까지 본 학생 비율)
  • 인기 강의 순위

기간 선택

화면 상단에서 조회 기간을 선택하여 원하는 기간의 통계를 확인할 수 있습니다.

자주 묻는 질문

관리 중 자주 발생하는 문제와 해결 방법입니다

Q1. 관리자 페이지에 접속할 수 없어요

A: 다음을 확인해 주세요:

  1. 관리자 계정으로 로그인했는지 확인하세요
  2. 웹사이트 주소 뒤에 /admin을 붙여 접속하세요
  3. 브라우저 캐시를 삭제하고 다시 로그인해 보세요
  4. 여전히 안 되면 개발팀에 연락하여 관리자 권한을 확인받으세요

Q2. 영상이 재생되지 않아요

A: 다음을 확인해 주세요:

  1. 입력한 Vimeo URL이 올바른지 확인하세요
  2. Vimeo에서 해당 영상이 삭제되지 않았는지 확인하세요
  3. Vimeo 영상의 공개 설정이 "Anyone" 또는 "Unlisted"인지 확인하세요
  4. "Private" 영상은 재생되지 않을 수 있습니다

Q3. 무통장입금을 승인했는데 학생이 강의를 볼 수 없대요

A: 다음을 시도해 주세요:

  1. 결제 관리 페이지에서 해당 결제 상태가 "완료"인지 확인하세요
  2. 학생에게 로그아웃 후 다시 로그인하도록 안내하세요
  3. 브라우저 새로고침을 해보도록 안내하세요
  4. 여전히 안 되면 개발팀에 문의하세요

Q4. 학생이 기기를 2개 넘게 등록했다고 해요

A: 다음과 같이 처리하세요:

  1. 회원 관리에서 해당 학생을 찾습니다
  2. 회원 상세 페이지에서 "등록 기기" 섹션을 확인합니다
  3. 사용하지 않는 기기를 삭제해 줍니다
  4. 학생이 다시 로그인하여 새 기기를 등록할 수 있습니다

Q5. 공지사항이 학생들에게 보이지 않아요

A: 다음을 확인하세요:

  1. 공지사항을 "저장"이 아닌 "게시"로 완료했는지 확인하세요
  2. 학생 계정으로 로그인하여 직접 확인해 보세요
  3. 브라우저 캐시 문제일 수 있으니 새로고침을 해보세요

Q6. 강의를 삭제했는데 복구할 수 있나요?

A: 죄송하지만 삭제된 강의는 복구할 수 없습니다. 삭제하기 전에 반드시 확인하시고, 필요 없는 강의는 삭제보다는 "비공개"로 전환하는 것을 권장합니다.

Q7. 강의 가격을 변경하면 이미 구매한 학생들에게도 영향이 있나요?

A: 아니요. 가격을 변경해도 이미 구매한 학생들에게는 영향이 없습니다. 변경된 가격은 앞으로 구매하는 학생들에게만 적용됩니다.

Q8. 환불을 승인했는데 취소할 수 있나요?

A: 환불 승인 후 "완료 처리"를 하기 전이라면 개발팀에 문의하여 상태를 변경할 수 있습니다. 단, "완료 처리"까지 완료된 경우에는 학생이 다시 강의를 구매해야 합니다.

운영 팁

효과적인 플랫폼 운영을 위한 유용한 팁입니다

효과적인 강의 운영

신규 강의 홍보하기

  • 첫 번째 영상을 미리보기로 설정하여 강의 맛보기 제공
  • 공지사항에 신규 강의 오픈 소식 게시
  • 상단 고정 공지로 설정하여 많은 학생이 볼 수 있게 하기

학생 만족도 높이기

  • 1:1 문의에 빠르게 답변 (24시간 이내 권장)
  • 정기적으로 공지사항을 올려 소통
  • 강의 자료를 충실히 제공
  • 영상 순서를 논리적으로 배치

매출 향상시키기

  • 대시보드에서 인기 강의 분석
  • 완강률이 낮은 강의는 내용 개선
  • 정기적으로 신규 강의 오픈

정기적으로 확인할 사항
